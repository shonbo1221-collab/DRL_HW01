# Personal Portfolio | Tsung-Sheng Yu

## Overview
This is a modern, premium personal digital business card / landing page developed for **Tsung-Sheng Yu**. The project features a high-quality frontend design, emphasizing a clean aesthetic and smooth user experience.

**Theme:** Ocean Theme 🌊 (Deep navy background with Cyan and Teal dynamic lighting)

## Features

### 1. Web Development
- **Main Interface (`index.html`)**: A sleek, single-page layout featuring a dynamic greeting, current time display, and a professional "Online" status indicator.
- **Styling (`style.css`)**: Implemented a "Glassmorphism" design system using CSS variables, smooth gradients, and micro-animations for a premium look and feel.
- **Dynamic Features**: Added JavaScript to handle real-time clock updates (date and time) and responsive entrance animations.
- **Responsive Design**: Fully optimized for mobile screens, adjusting typography and spacing dynamically.

### 2. File Structure
- `index.html`: Core structure, content, and JavaScript time logic.
- `style.css`: Visual design, theming, and CSS keyframe animations.
- `README.md`: This project documentation.

## Visual Design Details
- **Colors**: Deep ocean navy background (`#020c1b`), with Sky Blue (`#0ea5e9`), Sea Green (`#2dd4bf`), and Ocean Blue (`#3b82f6`) accent blobs.
- **Typography**: Employs Google Fonts (`Outfit` for headings/digits, `Inter` for body text) to ensure a modern, readable interface.
- **Effects**: Heavy backdrop blur for glassmorphism, dynamic glowing text shadows, and floating background animations.
